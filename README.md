# LibraReminder

このアプリは、神戸市立図書館で借りた本の貸出期限を、googleカレンダーへ転送・更新するコマンドラインプログラムです。

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'libra_reminder'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install libra_reminder


    $ sudo gem install google-api-client -v 0.6.4

- ブラウザを起動してAPIを登録
